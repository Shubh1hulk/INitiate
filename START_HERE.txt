🚀 QUICK START (5 MINUTES)
───────────────────────────────────────────────────────────────────────────────

1. INSTALL DEPENDENCIES

   cd backend
   npm install

   cd ../frontend
   npm install

2. SETUP ENVIRONMENT

   Backend:
   cd backend && cp .env.example .env

   Frontend:
   cd frontend && echo "NEXT_PUBLIC_API_URL=http://localhost:5000/api" > .env.local

3. GET GROQ API KEY (FREE)

   Visit: https://console.groq.com
   Sign up (free)
   Create API key
   Add to backend/.env as GROQ_API_KEY

4. START SERVERS (Use 2 terminals)

   Terminal 1 (Backend):
   cd backend && npm run dev
   → http://localhost:5000

   Terminal 2 (Frontend):
   cd frontend && npm run dev
   → http://localhost:3000

5. OPEN APP

   Visit: http://localhost:3000
   Register or login
   Start using!

🆘 TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

Backend won't start?
→ Check MongoDB is running
→ Verify GROQ_API_KEY in .env
→ Check port 5000 is not in use

Frontend won't connect?
→ Ensure backend is running
→ Check NEXT_PUBLIC_API_URL in .env.local
→ Frontend needs to be on http://localhost:3000

API errors?
→ Check backend console for details
→ Verify all env variables are set
→ Check network tab in browser dev tools

See README.md or QUICK_START.md for more help
